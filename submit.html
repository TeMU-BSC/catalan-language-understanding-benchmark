Download sample submission files

See the sample files for a hint on naming them.

Make sure that each prediction file is named according to the following:

  • NER: ancora-ca-ner_predictions.txt
  • POS: pos_predictions.txt
  • STS-ca: sts-ca_predictions.txt
  • TeCla: tecla_predictions.txt
  • TE-Ca: teca_predictions.txt
  • CatalanQa: catalanqa_predictions.json
  • XQuAD-ca: xquad-ca_predictions.json
  • Massive: massive_predictions.txt
